Opis Projekta

Spletna trgovina je moderna e-commerce aplikacija, razvita z uporabo najnovejših spletnih tehnologij. Projekt vključuje React frontend, Node.js/Express backend, MySQL bazo podatkov in napredni CI/CD pipeline z Docker kontejnerizacijo.

Aplikacija omogoča uporabnikom pregledovanje izdelkov, dodajanje v košarico, registracijo, prijavo in nakupovanje čevljev z varnim plačilnim sistemom.

Tehnološki Sklad

React 18 Node.js Express.js MySQL Docker GitHub Actions Vercel Docker Hub Jest Testing Nginx

CI/CD Pipeline

Projekt vključuje napredni CI/CD pipeline z GitHub Actions, ki avtomatično:

  • Izvaja teste za frontend in backend
  • Gradi aplikacijo z optimiziranim caching-om
  • Ustvarja Docker slike in jih objavlja na Docker Hub
  • Deploya aplikacijo na Vercel
  • Objavlja statično stran na GitHub Pages
  • Upravlja Development in Production okolja
